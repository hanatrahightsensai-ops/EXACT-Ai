# EXACT AI

Small single-file frontend that generates adaptive physics quizzes (Thai/English UI).

Files:
- `exact-ai.html` — main single-file app (open in browser).

Quick start:
1. Open `exact-ai.html` in your browser (double-click or `File → Open`).
2. Enter a nickname, select grade and goal, then choose a topic and start the quiz.

Notes:
- No build step required — static HTML/CSS/JS only.
- To publish, push this repository to GitHub (already set up in this workspace).
